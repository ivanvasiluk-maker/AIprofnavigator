# Test Cases Before Release

## RU tests

### 1. NextYou
Input: "я дурак если не знаю квантовую физику"

Expected:
- `core_thought` сохраняет исходную мысль.
- `fact` = "я не знаю квантовую физику".
- Не должно быть формулировки "Пользователь считает...".
- Ответ короткий.

### 2. Aristotle
Input: "я тупой не могу выучить стих"

Expected:
- Цель = выучить стих.
- Первый шаг = первая строка 3 раза.
- Не должно быть формулировки "улучшить самооценку".

### 3. Marcus Aurelius
Input: "боюсь что меня заменит ИИ"

Expected:
- Не в твоей власти = рынок/скорость изменений.
- В твоей власти = сегодняшний эксперимент.
- Не должно быть generic: "ИИ инструмент, не враг".

### 4. Diogenes
Input: "хочу смотреть порно вместо работы"

Expected:
- Не морализирует.
- Называет быстрый побег.
- Дает 10 минут не кормить импульс.

### 5. High arousal
Input: "сижу дико напряжённая и нервная"

Expected:
- Не пишет "расслабься".
- Дает телесный шаг.
- Длина ответа: 350-500 символов максимум.
