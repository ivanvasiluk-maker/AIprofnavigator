# Career Bot Mandatory Prelaunch Cases

Перед широким запуском бот обязан пройти 20 кейсов из файла `tests/quality_gate_cases.json`.

## Обязательные кейсы

1. Рабочий после миграции без языка.
2. IT-специалист с хорошим английским.
3. Медсестра с непризнаваемым дипломом.
4. Учительница с ребенком и ограничением по времени.
5. Человек после выгорания.
6. Человек в сильной тревоге.
7. Человек без резюме.
8. Человек с резюме, но без понимания рынка.
9. Человек, который хочет открыть бизнес.
10. Человек, который хочет только стабильную работу.
11. Человек с противоречивыми ответами.
12. Человек, который отвечает не на тот вопрос.
13. Человек, который все пропускает.
14. Человек, который пишет 2 слова.
15. Человек, который присылает очень длинную историю.
16. Человек, который меняет цель по ходу теста.
17. Человек, который говорит "не знаю".
18. Человек, который не согласен с маршрутом.
19. Человек, который нажимает старые кнопки во время генерации.
20. Человек, у которого ломается PDF.

## Чеклист для каждого кейса

- Факты не выдуманы.
- Противоречия замечены.
- Маршрут объяснен.
- Первый шаг реалистичный.
- Есть понятное следующее действие.
- Бот не зациклился.
- PDF или HTML отдан пользователю.

## Автоматическая проверка

Запуск:

`.\.venv\Scripts\python.exe -m unittest tests.test_quality_gate -v`

Проверяется:

- кейсов ровно 20,
- `id` от 1 до 20 без пропусков,
- для каждого кейса заполнен обязательный чеклист из 7 пунктов.
